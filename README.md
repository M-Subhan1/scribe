# Scribe - PDF to Word CLI
Scribe cli enables users to convert any given pdf file to word document. To support broad usage, we extract images from each page, feed it to an OpenAI model to get back the transcribed text as markdown. The markdown file is, then, processed to generate word documents.

## Usage
- Install the package.
- run `scripe init` to set-up required keys
- run `scribe help` to see the help document
